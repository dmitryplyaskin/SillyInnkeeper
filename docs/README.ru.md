# SillyInnkeeper

**SillyInnkeeper** — это приложение для удобной работы с карточками персонажей для SillyTavern. Оно предоставляет мощные инструменты для поиска, фильтрации и сортировки тысяч карточек, а также интеграцию с SillyTavern для быстрого запуска персонажей.

> 🔗 **Расширение для SillyTavern**: [ST-Extension-SillyInnkeeper](https://github.com/dmitryplyaskin/ST-Extension-SillyInnkeeper) — установите это расширение в SillyTavern для интеграции с SillyInnkeeper.

> 📢 **Telegram канал**: [@SillyInnkeeper](https://t.me/SillyInnkeeper) — новости, обновления и обсуждения проекта.

## 💡 Зачем нужен SillyInnkeeper?

Если у вас большая коллекция карточек персонажей (сотни или тысячи файлов), вы наверняка сталкивались с проблемами при работе с ними:

- **Лаги и зависания** — встроенные инструменты SillyTavern не справляются с большим объемом карточек
- **Медленный поиск** — поиск по карточкам занимает много времени или вообще не работает
- **Неудобная фильтрация** — ограниченные возможности фильтрации и сортировки
- **Сложность навигации** — трудно найти нужную карточку среди тысяч файлов
- **Отсутствие метаданных** — сложно понять, что находится в карточке, не открывая её

SillyInnkeeper решает все эти проблемы, предоставляя быстрый и удобный способ работы с вашей коллекцией карточек.

## ✨ Основные возможности

### Полноценная интеграция с SillyTavern

- **Просмотр и редактирование карточек**: Полный доступ к содержимому карточек персонажей из SillyTavern
- **Управление чатами**: Просмотр информации о чатах, связанных с карточками, фильтрация по профилям пользователей
- **Синхронизация данных**: Автоматическая синхронизация метаданных между SillyInnkeeper и SillyTavern
- **Запуск одним кликом**: Кнопка "Play" мгновенно импортирует карточку в SillyTavern
- **Автоматический импорт**: Расширение для SillyTavern автоматически получает карточки из SillyInnkeeper

### Мощный поиск и фильтрация

- **Поиск по имени**: Быстрый поиск карточек по названию персонажа
- **Текстовый поиск**: Полнотекстовый поиск по содержимому карточек с поддержкой двух режимов:
  - Режим "LIKE" — простой поиск по подстроке
  - Режим "FTS" — полнотекстовый поиск с ранжированием по релевантности
  - Поиск по полям: описание, личность, сценарий, первое сообщение, примеры сообщений, заметки создателя, системные промпты, инструкции истории, альтернативные приветствия, групповые приветствия
- **Фильтрация по метаданным**:
  - По создателю (один или несколько)
  - По версии спецификации (V1/V2/V3)
  - По тегам (множественный выбор)
  - По паттернам (настраиваемые правила)
- **Фильтрация по дате**: Диапазон дат создания карточек
- **Фильтрация по размеру**: Поиск по примерному количеству токенов промпта (мин/макс)
- **Фильтрация по наличию полей**: Поиск карточек с определенными полями или без них:
  - Заметки создателя
  - Системные промпты
  - Инструкции истории
  - Личность
  - Сценарий
  - Примеры сообщений
  - Character Book (lorebook)
- **Фильтрация по альтернативным приветствиям**: Наличие и минимальное количество
- **Фильтрация по источнику**: Карточки из SillyTavern, из папки или все
- **Фильтрация по статусу**: Скрытые, избранные карточки
- **Фильтрация по чатам SillyTavern**:
  - Количество чатов (равно/больше/меньше)
  - Профили пользователей
  - Скрытие карточек без чатов
- **Гибкая сортировка**:
  - По дате создания (новые/старые первыми)
  - По имени (А-Я / Я-А)
  - По количеству токенов промпта (по убыванию/возрастанию)
  - По количеству чатов SillyTavern (по убыванию/возрастанию)
  - По дате последнего чата (новые/старые первыми)
  - По дате первого чата (новые/старые первыми)
  - По релевантности (при использовании полнотекстового поиска)

### Управление библиотекой карточек

- **Автоматическое сканирование**: Просто укажите папку с карточками — приложение само найдет и проиндексирует все PNG-файлы
- **Автоматическое обновление**: При добавлении новых карточек или изменении существующих приложение автоматически обновит информацию
- **Работа с дубликатами**: Автоматическое обнаружение одинаковых карточек и удобное управление ими

### Удобный просмотр карточек

- **Вся информация в одном месте**: Просмотр всех данных карточки без необходимости открывать файл
- **Организация по вкладкам**: Информация разделена на категории для удобства:
  - Основная информация (имя, описание, личность, сценарий, первое сообщение)
  - Альтернативные приветствия
  - Системные промпты
  - Полный JSON (для продвинутых пользователей)
- **Просмотр изображений**: Увеличение изображений карточек с возможностью размытия (цензура)
- **Метаданные**: Видно ID, версию спецификации, дату создания и другую полезную информацию
- **Экспорт карточек**: Скачивание PNG-файлов с корректными метаданными для использования в других приложениях

### Удобство использования

- **Быстрая работа**: Приложение оптимизировано для работы с тысячами карточек без лагов
- **Автоматические миниатюры**: Все карточки отображаются с миниатюрами для быстрого просмотра
- **Темы**: Светлая, темная и автоматическая тема (следует системным настройкам)
- **Два языка**: Поддержка русского и английского языков

### Поддержка всех форматов

- Поддержка Character Card V1, V2 и V3 — работайте с карточками любого формата

## 💻 Системные требования

- **Node.js**: версия 18.x или выше (рекомендуется 20.x или 24.x)
- **Yarn**: версия 4.12.0 (или npm 9.x+)
- **Операционная система**: Windows 10/11, Linux, macOS

## 📦 Установка

### Предварительные требования

Убедитесь, что у вас установлены:

- [Node.js](https://nodejs.org/) (версия 18.x или выше)
- [Yarn](https://yarnpkg.com/) (версия 4.12.0) или npm

### Способ 1: Автоматическая установка (Windows)

1. Клонируйте репозиторий:

```bash
git clone https://github.com/dmitryplyaskin/SillyInnkeeper.git
cd SillyInnkeeper
```

2. Запустите скрипт установки:

```bash
start.bat
```

Скрипт автоматически установит все зависимости, соберет проект и запустит сервер. Браузер откроется автоматически.

### Способ 2: Ручная установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/dmitryplyaskin/SillyInnkeeper.git
cd SillyInnkeeper
```

2. Установите зависимости для сервера:

```bash
cd server
yarn install
# или
npm install
```

3. Установите зависимости для клиента:

```bash
cd ../client
yarn install
# или
npm install
```

4. Соберите проект:

Сборка клиента:

```bash
cd client
yarn build
# или
npm run build
```

Сборка сервера:

```bash
cd server
yarn build
# или
npm run build
```

5. Запустите сервер:

```bash
cd server
yarn start
# или
npm start
```

6. Откройте браузер и перейдите по адресу:

```
http://127.0.0.1:48912
```

### Способ 3: Установка на Android/Termux

1. Установите Termux из [F-Droid](https://f-droid.org/packages/com.termux/) (версия из Google Play устаревшая).

2. Установите необходимые пакеты:

```bash
pkg update && pkg upgrade
pkg install nodejs-lts git binutils build-essential python libvips imagemagick
```

> **Для чего нужны эти пакеты:**
> - `nodejs-lts` — среда выполнения Node.js
> - `git` — клонирование репозитория
> - `binutils`, `build-essential` — компилятор C++ (для сборки нативных модулей)
> - `python` — требуется для `node-gyp` (используется `better-sqlite3` и `sharp`)
> - `libvips` — библиотека обработки изображений (требуется `sharp`)
> - `imagemagick` — поддержка дополнительных форматов изображений (требуется `libvips`)

3. Примените gyp fix (предотвращает ошибки сборки нативных модулей):

```bash
mkdir -p ~/.gyp && echo "{ 'variables': { 'android_ndk_path': '' } }" > ~/.gyp/include.gypi
```

4. Клонируйте репозиторий:

```bash
git clone https://github.com/dmitryplyaskin/SillyInnkeeper.git
cd SillyInnkeeper
```

5. Установите зависимости сервера:

```bash
cd server
npm install
```

6. Установите зависимости клиента:

```bash
cd ../client
npm install
```

7. Соберите клиент:

```bash
cd ../client
npm run build
```

8. Запустите сервер:

```bash
cd ../server
npm start
```

9. Откройте браузер и перейдите по адресу:

```
http://127.0.0.1:48912
```

> ⚠️ **Примечание**: На Android скрыта иконка выбора папки, так как системные диалоги недоступны из контекста сервера/браузера. Введите полный путь вручную (например, `/storage/emulated/0/Download/characters`).

### Конфигурация (.env)

Можно настроить **хост/порт** и интеграцию с SillyTavern через корневой файл `.env`.

- Скопируйте [`env.example`](../env.example) в `.env` (в корне репозитория, рядом с `package.json`)
- Переменные:
  - `INNKEEPER_HOST` — интерфейс биндинга (по умолчанию `127.0.0.1`). Для доступа из локальной сети используйте `0.0.0.0`.
  - `INNKEEPER_PORT` — порт сервера (по умолчанию `48912`)
  - `ST_PORT` — порт SillyTavern для дефолтного CORS allowlist (по умолчанию `8000`)
  - `CORS_ALLOW_ORIGINS` — дополнительные разрешённые origin'ы (через запятую), например `http://192.168.1.50:8001`

## 🚀 Быстрый старт

### Первый запуск

1. **Запустите приложение** (см. раздел [Установка](#-установка))

2. **Настройте путь к папке с карточками**:

   - Откройте настройки (кнопка "Настройки" в верхней панели)
   - Укажите путь к папке, где хранятся ваши PNG-файлы карточек
   - Сохраните настройки

3. **Дождитесь завершения сканирования**:

   - При первом запуске приложение автоматически начнет сканирование указанной папки
   - Прогресс сканирования отображается в интерфейсе
   - После завершения все карточки будут доступны для поиска и просмотра

4. **Начните использовать**:
   - Используйте поиск и фильтры для нахождения нужных карточек
   - Кликните на карточку для просмотра детальной информации
   - Используйте кнопку "Play" для запуска карточки в SillyTavern (если настроена интеграция)

### Настройка интеграции с SillyTavern

1. Установите расширение [ST-Extension-SillyInnkeeper](https://github.com/dmitryplyaskin/ST-Extension-SillyInnkeeper) в SillyTavern

2. В настройках расширения укажите URL SillyInnkeeper: `http://127.0.0.1:48912`

   Если SillyTavern открыт на другой машине, используйте IP/hostname машины с SillyInnkeeper (например `http://192.168.1.10:48912`).

3. Включите "Auto-connect" для автоматического подключения

4. Теперь вы можете использовать кнопку "Play" в SillyInnkeeper для автоматического импорта карточки в SillyTavern

## 📖 Использование

### Основной интерфейс

Главный экран приложения состоит из:

- **Верхняя панель**: Заголовок, переключатель темы, настройки вида, кнопки настроек и фильтров
- **Сетка карточек**: Список всех карточек с миниатюрами
- **Боковая панель фильтров**: Открывается по кнопке "Фильтры"

### Поиск и фильтрация

1. **Откройте панель фильтров** (кнопка "Фильтры" в верхней панели)

2. **Используйте поиск**:

   - **Поиск по имени**: Введите название персонажа для быстрого поиска
   - **Текстовый поиск**: Введите текст для поиска по содержимому карточек
     - Выберите режим поиска: "LIKE" (простой поиск) или "FTS" (полнотекстовый поиск с ранжированием)
     - Выберите поля для поиска: описание, личность, сценарий, первое сообщение, примеры сообщений, заметки создателя, системные промпты, инструкции истории, альтернативные приветствия, групповые приветствия

3. **Примените фильтры**:

   - **Метаданные**: Создатель, версия спецификации (V1/V2/V3), теги, паттерны
   - **Дата создания**: Укажите диапазон дат
   - **Токены промпта**: Минимальное и максимальное количество токенов
   - **Альтернативные приветствия**: Наличие и минимальное количество
   - **Наличие полей**: Выберите поля, которые должны присутствовать/отсутствовать (заметки создателя, системные промпты, инструкции истории, личность, сценарий, примеры сообщений, Character Book)
   - **Источник**: Все карточки, только из SillyTavern, только из папки
   - **Статус**: Скрытые, избранные карточки
   - **Чаты SillyTavern**: Количество чатов (равно/≥/≤), профили пользователей, скрытие карточек без чатов

4. **Выберите сортировку**: По дате создания, имени, количеству токенов, количеству чатов, дате последнего/первого чата или по релевантности (при полнотекстовом поиске)

5. **Результаты обновляются автоматически** при изменении фильтров

6. **Сбросьте фильтры**: Используйте кнопку "Сбросить" для очистки всех фильтров

### Просмотр карточки

1. **Кликните на карточку** в сетке для открытия детальной информации

2. **Изучите информацию**:

   - **Вкладка "Основная"**: Основная информация о персонаже
   - **Вкладка "Альтернативные приветствия"**: Все альтернативные приветствия
   - **Вкладка "Системные"**: System prompt и post history instructions
   - **Вкладка "Сырой JSON"**: Полный JSON карточки для редактирования

3. **Используйте действия**:
   - **Play**: Запустить карточку в SillyTavern
   - **Скачать**: Скачать PNG-файл карточки
   - **Переименовать**: Изменить имя основного файла
   - **Удалить**: Удалить карточку или дубликат

## 🔗 Интеграция с SillyTavern

SillyInnkeeper интегрируется с SillyTavern через расширение [ST-Extension-SillyInnkeeper](https://github.com/dmitryplyaskin/ST-Extension-SillyInnkeeper).

### Установка расширения

1. Откройте SillyTavern
2. Перейдите в **Extensions → Extension Installer**
3. Вставьте URL репозитория:
   ```
   https://github.com/dmitryplyaskin/ST-Extension-SillyInnkeeper
   ```
4. Нажмите "Install"

### Настройка расширения

1. Откройте **Extensions → SillyInnkeeper** в SillyTavern
2. Укажите **SillyInnkeeper URL**: `http://127.0.0.1:48912` (или ваш порт)
3. Включите **Auto-connect** (рекомендуется)
4. Опционально включите **Report import result back to SillyInnkeeper**
5. Опционально включите **Open imported character** для автоматического открытия импортированного персонажа

### Использование

1. Откройте карточку в SillyInnkeeper
2. Нажмите кнопку **"Play"** в детальном просмотре
3. Карточка автоматически импортируется в SillyTavern
4. Если включена опция "Open imported character", персонаж откроется автоматически
